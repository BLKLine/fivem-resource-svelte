# FiveM Resource

## Description

A FiveM typescript resource that includes a Svelte HTML page

## Scripts

Builds FiveM Scripts and Svelte

```bash
npm run build
```

Builds Svelte

```bash
npm run build:html
```

Build FiveM scripts

```bash
npm run build:scripts
```

Live Updating - FiveM Scripts and Svelte

```bash
npm run dev
```

Live Updating Svelte

```bash
npm run dev:html
```

Live updating FiveM Scripts

```bash
npm run dev:scripts
```
